# Experimental Data Analysis (Python Learning Portfolio)

This repository contains simple, beginner‑friendly examples of analysing and integrating experimental datasets using Python. As I continue developing my computational skills, I am focusing on building confidence in:

- basic scripting and workflow organisation  
- loading and cleaning experimental datasets  
- simple statistical analyses  
- visualisation of results (e.g., line plots, bar graphs, scatter plots)  
- integrating multiple readouts to support experimental interpretation  

The examples here reflect my current learning journey and interest in data‑driven therapeutic discovery. All scripts are intentionally clear and lightweight, and they demonstrate how I am building foundational skills in Python‑based scientific analysis.
